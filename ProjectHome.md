GPS data logging midlet with extra features for Java MIDP2 Phones that support <a href='http://en.wikipedia.org/wiki/JSR82'>JSR-82 (Bluetooth API)</a>, <a href='http://en.wikipedia.org/wiki/JSR-75'>JSR-75 (Filesystem access)</a> and <a href='http://en.wikipedia.org/wiki/Location_API_for_Java_ME'>JSR-179 (Location API)</a>.
